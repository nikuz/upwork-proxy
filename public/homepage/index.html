Upwatcher

Do you want to work on a paradise beach or at any pretty place? Just install application Upwatcher, which helps you conveniently search, filter, and save interesting jobs by the world's biggest platform for freelancers upwork.com (elance.com).

The app allows you to stay in touch with thousands clients who post new interesting work proposals.

Upwatcher based on native Upwork API. Application haven't access to any you data on Upwork.com, also don't change any settings. It's working locally, API access needed only for use native Upwork search.

Features:
- customizable search
- ability to mark jobs as Favorites
- show system Notifications
